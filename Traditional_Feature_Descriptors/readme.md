Distibuted Reverse Image Search Engine

Setup Instructions:
pip install opencv-python
pip install opencv-contrib-python
pip install numpy
pip install scikit-image
pip install pillow
pip install tqdm

How to run:
python Traditional Features Descriptors.py

Then it shows 2 options:
1: Build feature database
2: Search similar images

option 1 will build feature database
option 2 requires a query. 
Example of query: dataset/Weather_Dataset/fogsmog/4075.jpg
